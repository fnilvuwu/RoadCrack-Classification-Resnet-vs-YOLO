About Website

Website Purpose:
This website aims to evaluate the output of performance testing of object detection models Residual Neural Network (ResNet-50) and YOLOv8 in road crack classification using the mAP metric. Through this website, users can compare the accuracy and inference speed of both models in detecting and classifying road conditions (Good, Pothole, Crack).
